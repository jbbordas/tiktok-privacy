Politique de confidentialité

Dernière mise à jour : 5 avril 2025

1. Informations collectées

Cette application ne collecte aucune donnée personnelle. Elle est utilisée uniquement pour générer et uploader des vidéos horoscope sur TikTok.

2. Utilisation des données

Aucune donnée n’est stockée, partagée ou vendue. Les textes sont générés localement, et seules les vidéos sont envoyées à TikTok via leur API.

3. API TikTok

L’application utilise l’API TikTok pour uploader des vidéos dans les brouillons. Aucune publication automatique n’est effectuée.

4. Contact

Pour toute question, contactez-nous à : contact@exemple.com
